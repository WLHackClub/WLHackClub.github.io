# WL Hack Club Website

The WL Hack Club Website is made using Jekyll, a static site generator in Ruby.

## Deployment

Self-test code locally. Make sure Ruby 2.5.0 or later is installed.

- `bundler install`
- `bundler exec jekyll serve`
