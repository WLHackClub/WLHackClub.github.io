Workshops

Competitive Programming

Learn the fundamentals of competitive programming through algorithmic problem solving, coding strategy, and contest-style challenges. This workshop is a great way to sharpen your logic and write faster, more efficient solutions.

Game Development

This workshop is designed for beginners to dive into the basics of game development and bring their creative ideas to life. The skills learned in this workshop can be used to create a project at this hackathon. In 21st century, knowing how to code is a basic skill that everyone should learn and this workshop will allow you to start your journey into becoming a programmer.

3D Printing

Learn the basics of 3D printing and how to create your own 3D models. This workshop will cover the fundamentals of 3D design, giving you the basic skills to bring your ideas to life. Whether you're interested in creating prototypes or projects for this hackathon, this workshop will provide you with the foundational knowledge to get started.
