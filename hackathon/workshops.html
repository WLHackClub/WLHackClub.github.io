Workshops

3D Printing

Learn the fundamentals of 3D printing and watch your ideas come to life! Each participant will create their own 3d print to take home.

AI Game

Learn how to incorporate AI into a machine learning model with no code or math required! Machine learning has become increasingly popular in the last few years and the skills learned in this workshop can be transferred to real models.

Python

This workshop is designed for new programmers to learn the syntax of python and write their first programs. The skills learned in this workshop can be used to create a project at this hackathon, but in 21st century, knowing how to code is a basic skill that everyone should learn and this workshop will allow you to start your journey into becoming a programmer.

Computer Architecture

Want to learn how to build a custom PC? With the introduction to all the components of a PC, this workshop will help set you up to be able to personally configure all the parts to your custom PC, as well as to understand them thoroughly. CPU, RAM, power supply, and others, we've got a lot to show you!
